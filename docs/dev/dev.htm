You've reached the construction scaffolding for /dev, the nfoWare software-development tree.  This tree contains the current snapshot of the stable packages for open-source nfoWare tools, utilities, and samples.

  • The tree is also maintained and archived by the lead developer using Visual Source Safe. 
  • The material posted here is also archived in Visual Source Safe as part of configuration management for the nfoWare web site.  Many of the version-control "labels" in these files reflect that archiving.

This tree provides an appropriate organization for multiple software projects.  It is inspired by the same effort for ActiveODMA.  The development effort can be "shadowed," tested, and confirmed by using a directory tree that mirrors this one:

%nfoDev% environment variable for locating the dev/ directory wherever it is mirrored:
dev/ root folder of the  projects development, primarily for projects using C and C++
   |- confirm/ confirmation experiments for demonstrating correct setup and test approaches
   |- core/ fundamental libraries and materials shared among all nfoWare projects
   |- nodist/ descriptions of non-redistributable components that must be obtained and installed by you
   |- ab/ development of an addbib utility for creating bibliographic references using spiral development
   |- tools/ collection of redistributable utilities from other sources/projects used for nfoWare

The tree section, nodist, is where the dependencies on non-redistributable proprietary software are fully accounted for.  The section provides information on the specific dependencies that such materials are used to satisfy.  There is also information on how the non-redistributable materials can be obtained and installed at no expense beyond that of entering into a license agreement and downloading from a web site.

This index.htm page is part of the engineering, construction management, maintenance, and infrastructure of the nfoWare.com site.  You are welcome to explore this and other construction-information pages.   You'll need your hard hat and safety shoes.

If you'd rather find answers to questions about the nfoWare development tree, go directly to the default entry of this section.   If that doesn't work, come back later.  The section is blocked off whenever the construction crew is blasting in the area.
