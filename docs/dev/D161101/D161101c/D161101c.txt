                   VCBIND 0.1.0 RELEASE (PRESERVATION)
                   -----------------------------------

This is the preservation copy of the VCbind 0.1.0 release on nfoTools.

This is not recommended for further use.  The materials make reference to
nfoWare.com locations that are no longer appropriate.  They are preserved
intact because of the hash and signature values which cannot be changed.

The preservation is strictly for historical purposes.
