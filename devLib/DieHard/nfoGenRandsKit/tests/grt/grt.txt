                       GENERATED RNG TEST TEMPORARY FILES
                       ==================================

    This folder is designed for receipt of generated RNG-specific test files
    for temporary usage.  The files are quite large and generated in the
    testing of individual RNGs.  The files can always be regenerated using
    the same test-file generation and do not need to be preserved beyond the
    running of tests on them.

MANIFEST

    grt.txt
        This description file on tests for proper operation of nfoGenRands
        customizations and confirmation of output when possible.

    nfoGenBIOa16756.txt
    nfoGenBIOb16756.txt
        Test results from GemTemps 0.1.0

        NOTE: Test (result) files with no extension are excluded from Git.
