    VCrayApp-0.0.0.txt

    rayApp.bat
        Under development batch file for compiling Native VC code
        using the rayApp cache/ and app/ setup

    .vscode
        VS Code folder information for any folder the VCrayApp.zip template
        is extracted into

    app\
        the folder in which compiled executables and recourses are
        brought into

    cache\
        folder for rayApp-required assets and the caching of raylib
        modules for linking into the project .exe

    src\
        the recommended location for source-code of a project based on the
        VCrayApp template.
